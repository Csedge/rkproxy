### RKPROXY

rkproxy是一个 Shadowsocks/TCP/HTTPReproxy 代理软件


### 特性

#### shadowsocks

- 基于UDP实现的数据传输，加快网络环境较差时传输速度
- 只需一个程序，通过配置即可实现客户端-服务端
- 目前支持XOR、salsa20的加密传输（XOR最快）

#### TCP

- TCP数据代理

#### HTTP反向代理

- 简单易用的HTTP反向代理
